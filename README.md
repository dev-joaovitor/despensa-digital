# Despensa Digital
Projeto de engenharia de software em grupo.

# Tecnologias
## UI
- [Svelte](https://svelte.dev/)

## API
- [Golang](https://go.dev/)
  - HTTP - [go-chi/chi](https://github.com/go-chi/chi)
  - Cache - [redis/go-redis](https://github.com/redis/go-redis)
  - Database - [jackc/pgx](https://github.com/jackc/pgx)
  - Migrations - [golang-migrate](https://github.com/golang-migrate/migrate)
  - Authentication - bcrypt, [scs](https://github.com/alexedwards/scs)

## Database
- [PostgreSQL](https://www.postgresql.org/)
- [Redis](https://redis.io/)

## Infra
- [Docker](https://www.docker.com/)
